## Run Locally

Clone the project

```bash
  git clone https://github.com/marc-marquez/sneakers-sveltekit-app.git
```

Go to the project directory

```bash
  cd sneakers-sveltekit-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
